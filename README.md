# اجرای سرویس‌ها

برای `build` کردن و اجرای سرویس‌ها از دستور زیر استفاده می کنیم:

```bash
docker-compose up --build
```
این دستور باعث ساخت ایمیج‌ها طبق تعریف docker-compose.yml و اجرای کانتینرهای مرتبط با آن می شود.
